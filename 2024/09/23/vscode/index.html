VsCode windows 快捷键

Aaron2024-09-232024-09-26

常规

  • Ctrl+Shift+P, F1：显示命令面板
  • Ctrl+P：快速打开...
  • Ctrl+Shift+N：新窗口/实例
  • Ctrl+W：关闭窗口/实例
  • Ctrl+,：用户设置
  • Ctrl+K Ctrl+S：键盘快捷键

基本编辑

  • Ctrl+X：剪切（空选时剪切整行）
  • Ctrl+C：复制（空选时复制整行）
  • Alt+↑ / ↓：向上/向下移动行
  • Shift+Alt+↑ / ↓：复制向上/向下的行
  • Ctrl+Shift+K：删除行
  • Ctrl+Enter：在下面插入行
  • Ctrl+Shift+Enter：在上面插入行
  • Ctrl+Shift+\：跳转到匹配的括号
  • Ctrl+/：注释/取消注释行
  • Shift+Alt+A：块注释
  • Alt+Z：切换自动换行

导航

  • Ctrl+T：显示所有符号
  • Ctrl+P：快速打开文件
  • Ctrl+Shift+O：跳转到符号
  • Ctrl+G：跳转到行
  • F8：跳转到下一个错误或警告
  • Ctrl+Shift+Tab：导航活动组历史记录
  • Alt+← / →：前进/后退

搜索与替换

  • Ctrl+F：查找
  • Ctrl+H：替换
  • F3 / Shift+F3：查找下一个/上一个
  • Alt+Enter：选择查找结果中的所有出现项
  • Ctrl+D：将选择添加到下一个匹配项
  • Ctrl+K Ctrl+D：将下一个匹配项移到选择中
  • Alt+C / R / W：切换大小写/正则表达式/全词匹配

多光标与选择

  • Alt+Click：插入光标
  • Ctrl+Alt+↑ / ↓：在上/下插入光标
  • Ctrl+U：撤销上一个光标操作
  • Shift+Alt+I：在选择的行尾插入光标
  • Ctrl+I：选择当前行
  • Ctrl+Shift+L：选择所有出现的当前选择
  • Ctrl+F2：选择所有出现的当前单词
  • Shift+Alt+→ / ←：扩展/收缩选择
  • Ctrl+Shift+Alt+→ / ←：列选择（框选择）
  • Ctrl+Alt+Shift+PgUp / PgDn：列（框）选择页上/下

富语言编辑

  • Ctrl+Space：触发建议
  • Ctrl+Shift+Space：触发参数提示
  • Ctrl+Shift+F12：转到下一个问题
  • F12：转到定义
  • Alt+F12：在代码中显示定义
  • Ctrl+K F12：保持编辑器中的定义打开
  • Ctrl+.：快速修复
  • Shift+F12：转到引用
  • F2：重命名符号
  • Ctrl+K Ctrl+X：修剪尾部空白
  • Ctrl+M：切换 Tab 键作为焦点

文件管理

  • Ctrl+N：新文件
  • Ctrl+O：打开文件...
  • Ctrl+S：保存
  • Ctrl+Shift+S：另存为...
  • Ctrl+K S：全部保存
  • Ctrl+F4：关闭
  • Ctrl+K Ctrl+W：关闭所有
  • Ctrl+Shift+T：重新打开关闭的编辑器
  • Ctrl+K Enter：保持移动的编辑器打开
  • Ctrl+Tab：打开下一个
  • Ctrl+Shift+Tab：打开上一个
  • Ctrl+K P：复制活动文件的路径
  • Ctrl+K R：在资源管理器中显示
  • Ctrl+K O：在新窗口/实例中打开活动文件

显示

  • F11：切换全屏
  • Shift+Alt+0：切换编辑器布局（水平/垂直）
  • Ctrl+= / -：放大/缩小
  • Ctrl+B：切换侧边栏可见性
  • Ctrl+Shift+E：显示资源管理器/切换焦点
  • Ctrl+Shift+F：显示搜索
  • Ctrl+Shift+G：显示源控制
  • Ctrl+Shift+D：显示调试
  • Ctrl+Shift+X：显示扩展
  • Ctrl+Shift+H：替换文件
  • Ctrl+Shift+J：切换搜索详细信息
  • Ctrl+Shift+U：显示输出面板
  • Ctrl+K V：打开预览 Markdown 文件
  • Ctrl+K Z：禅模式（Esc 键退出）

调试

  • F9：切换断点
  • F5：开始/继续
  • Shift+F5：停止
  • F11 / Shift+F11：单步跳入/跳出
  • F10：单步执行
  • Ctrl+K Ctrl+I：显示悬停信息

集成终端

  • Ctrl+：显示集成终端
  • Ctrl+Shift+：创建新终端
  • Ctrl+C：复制选择
  • Ctrl+V：在活动终端中粘贴
  • Shift+PgUp / PgDn：滚动
  • Ctrl+Home / End：滚动到顶部/底部
